# AWS EC2 + Nginx Deployment

## 📌 Project Overview
Deployed a static website using Nginx on AWS EC2 instance.
Configured and tested the web server on both default HTTP port 80 and custom port 8080.

---

## ⚙️ Steps Performed
- Launched EC2 instance
- Configured Security Group (Port 22, 80 & 8080)
- Installed Nginx Web Server
- Started and enabled Nginx service
- Modified Nginx configuration to run on custom port 8080
- Deployed custom HTML page
- Tested access using:
  - http://<Public-IP>
  - http://<Public-IP>:8080

---

## 🛠 Technologies Used
- AWS EC2
- Linux (Amazon Linux)
- Nginx
- HTML & CSS

---

## 🌍 Access Methods

Default HTTP:
http://<Public-IP>

Custom Port:
http://<Public-IP>:8080
